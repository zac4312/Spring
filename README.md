This repository contains the Backend API for BlogsApp, an Android application that allows users to create and view blog posts.
Built with Spring Boot and connected to a MySQL database, it provides REST endpoints used by the Android client.

Features:
- Create blog posts (POST)
- Fetch all blog posts (GET)
- Filter through set genres(GET by Filter)
- Update blog posts (PUT)
- Delete blog posts (DELETE)
- JSON-based REST API
- Error handling + response messages

Tech Stack:
- Java 17
- Spring Boot
   - Spring Web
   - Spring Data JPA|JDBC
   - Maven
- REST API
- Android Studio
  - retrofit
